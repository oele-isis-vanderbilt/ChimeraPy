class OfflineCollector

Generic collector that stores only offline data streams.

The offline collector allows the use of both getitem and next to obtain the data pointer to a data stream to fetch the actual data.

Attributes:

  • data_streams (Dict[str, mm.OfflineDataStream]): A dictionary of the data streams that its keys are the name of the data streams.

  • global_timetrack (pd.DataFrame): A data frame that stores the time, data stream type, and data pointers to allow the iteration over all samples in all data streams efficiently.
