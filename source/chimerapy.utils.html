load_single_session(dir: pathlib.Path, verbose: bool = False) → Tuple[Sequence[chimerapy.core.data_stream.DataStream], pandas.core.frame.DataFrame]¶

Load a single Tobii session data.

If this is the first time loading the session, a longer processing will be done to convert gz data to csv. This is done so that future loads are in orders of magnitude faster.

Parameters

  • dir (pathlib.Path) – The session’s directory.

  • verbose (bool) – Debugging printout.

Returns

Tuple containing a list of the data streams (‘video’ and ‘gaze’) and the recording specifications as a data frame.

Return type

Tuple[Sequence[DataStream], pd.DataFrame]
